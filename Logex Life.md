# Zero‑Arity Predicates in Prolog: Metaphysics, Logic, and Trivial Truths

Zero‑arity predicates—facts like `Jane.` or `Civilization.`—are the smallest possible truth‑atoms in Prolog. They represent pure propositions without arguments, structure, or attributes. This article explains how they work, how they translate into trivial classical logic, and how they carry metaphysical meaning.

---

## 1. What Prolog Does With Statements Like `Jane.`

When you write:

```prolog
Jane.
Corporation.
Civilization.
```

Prolog interprets each line as a **0‑arity predicate**.  
These are propositions that are simply **true** in the logic universe.

Querying them:

```prolog
?- Jane.
```

returns:

```
true.
```

They are truth‑flags, not objects.

---

## 2. Classical Logic Translation ($\text{FOL}$)

Each Prolog fact corresponds to a propositional variable:

- `Jane.` → $Jane$
- `Corporation.` → $Corporation$
- `Civilization.` → $Civilization$

These are atomic truths:



$$
Jane = \text{True},\quad Corporation = \text{True},\quad Civilization = \text{True}.
$$



No quantifiers, no arguments—just truth.

---

## 3. GitHub‑Inline Logical Statements

Here is how the same mapping looks in GitHub Markdown:

- `Jane.` → $Jane$
- `Corporation.` → $Corporation$
- `Civilization.` → $Civilization$

```prolog

```

Logical translation:



$$
Exists \leftarrow Jane \land Civilization.
$$



Or in propositional form:

$Exists \equiv Jane \land Civilization$
